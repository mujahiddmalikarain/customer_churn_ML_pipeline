# customer_churn_ML_pipeline
# Churn ML Pipeline (Training → Threshold Tuning → FastAPI Deployment)

This project builds a production-style machine learning pipeline for customer churn prediction:
- Data ingestion from Excel
- Leakage-safe preprocessing
- Scikit-learn Pipeline + ColumnTransformer
- Baseline model training + metrics
- Threshold tuning (business decision tuning)
- FastAPI endpoint for inference
- Curl/Swagger testing
